# IndexedDB Examples

## Quick start

1. Install dependencies `npm i`
2. Start server
  - Usage: `node server <folder>`
  - Example: `node server Pragmatic`
3. Open in browser: [http://127.0.0.1:8000/](http://127.0.0.1:8000/)
